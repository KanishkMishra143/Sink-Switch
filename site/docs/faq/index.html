FAQ

Common questions and troubleshooting tips for Sink-Switch.

Audio switches, but current app doesn't follow?

On some older PulseAudio setups, active streams don't follow the default sink automatically. Sink-Switch tries to force this, but ensure "Move active streams" is enabled in your configuration.

Does it support PipeWire?

Yes! Sink-Switch is fully compatible with PipeWire. It uses pactl which PipeWire's pulse emulation layer handles perfectly.

Is there any latency?

The switch itself is near-instant (sub-50ms). Any delay you hear is typically the application (like Chrome or Spotify) buffering as it moves to the new device.

The tray icon is missing on Linux?

Ensure you have a system tray extension installed if you are using GNOME (like "AppIndicator and KStatusNotifierItem Support").

How can I contribute?

We love contributions! Check out our GitHub repository, open an issue, or submit a pull request.
